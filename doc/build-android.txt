- Build debug version:

    mkdir -p build
    cd build
    cmake -DVS_PLATFORM="android" ..
    make
    make apk

- Build release version:

    mkdir -p build
    cd build
    cmake -DVS_PLATFORM="android" -DCMAKE_BUILD_TYPE="MinSizeRel" -DVS_KEYCHAIN="[keychain URL or path]" -DVS_KEY_ALIAS="[Key alias]" -DVS_KEYCHAIN_PASSWORD="[Keychain password]" -DVS_KEY_PASSWORD="[Key password]" ..
    make
    make apk_release
