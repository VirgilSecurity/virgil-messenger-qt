- Build debug version:

    mkdir -p build
    cd build
    cmake -DVS_PLATFORM="macos" ..
    make
    make dmg_debug

- Build release version:

    mkdir -p build
    cd build
    cmake -DVS_PLATFORM="macos" -DCMAKE_BUILD_TYPE="Release"  -DVS_MACOS_IDENT="Ident for signing" ..
    make
    make dmg_release
